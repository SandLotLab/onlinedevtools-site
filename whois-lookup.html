About this tool

This WHOIS lookup tool uses RDAP to retrieve domain/IP registration metadata helpful for threat analysis and infrastructure attribution.

Common use cases

  • Investigate suspicious domains during incidents.
  • Validate ownership context for infrastructure assets.
  • Check network records for abuse investigations.

How it works

  1. Enter a domain or IPv4 address.
  2. Run lookup.
  3. Review RDAP output JSON.

FAQ

Why can lookup fail in browser?
Some RDAP providers block cross-origin browser requests.

Is this classic WHOIS protocol?
No, this tool uses modern RDAP endpoints.
