About this tool

This URL parser is built for debugging auth callbacks, redirects, API links, and tracking parameters. It helps you quickly inspect each URL segment without manual splitting.

Common use cases

  • Inspect OAuth callback query strings.
  • Debug malformed links and redirect paths.
  • Review UTM parameter values for analytics QA.

How it works

  1. Paste a full URL.
  2. Click Parse URL.
  3. Review parsed structure and query key/value pairs.

FAQ

Does this decode query parameters?
Yes, URLSearchParams output is decoded for readability.

Can I parse relative URLs?
This tool expects absolute URLs including protocol.
