About this tool

This tool helps quickly identify your current public IP and retrieve metadata for a target IPv4 address, which is useful for network troubleshooting and access verification.

Common use cases

  • Confirm public egress IP for firewall allowlists.
  • Investigate geolocation mismatch reports.
  • Validate ASN/network ownership at a glance.

How it works

  1. Find your IP automatically or enter an IPv4 manually.
  2. Run lookup request.
  3. Review returned metadata JSON.

FAQ

Will geolocation always be exact?
No, IP geolocation is approximate.

Is this private data collection?
No, this utility only shows lookup output and does not store your input.
